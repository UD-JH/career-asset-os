# Career Asset OS MVP v3

업무를 기록하고, 구조화하고, 자산으로 남기고, 커리어 문장으로 변환하는 개인용 웹앱입니다.

## 이번 버전에서 되는 것
- 업무 등록 / 수정 / 삭제
- 문제 / 행동 / 결과 / 배운 점 구조화
- 역량 태그 입력
- 자산 이름 / 유형 입력
- 이력서 bullet 생성
- 포트폴리오 문단 생성
- 면접 STAR 생성
- Skill Map 시각화
- 프로젝트 필터 / 역량 필터 / 검색
- Asset Library 전용 화면
- 월간 요약 자동 생성
- Markdown export
- localStorage 저장
- JSON 내보내기 / 불러오기
- 데모 데이터 넣기

## 실행 방법
1. 파일을 같은 폴더에 둡니다.
2. `index.html`을 브라우저로 엽니다.
3. 필요하면 GitHub 저장소에 올린 뒤 GitHub Pages로 배포합니다.
